
# Machine Learning Assignment 2

## Problem Statement
Implementation of multiple classification models on Breast Cancer dataset and deployment using Streamlit.

## Dataset Description
Breast Cancer Wisconsin Dataset (UCI Repository)
Instances: 569
Features: 30
Binary Classification Problem

## Models Implemented
- Logistic Regression
- Decision Tree
- KNN
- Naive Bayes
- Random Forest
- XGBoost

## Evaluation Metrics
- Accuracy
- AUC
- Precision
- Recall
- F1 Score
- MCC

Comparison table included in repository.
